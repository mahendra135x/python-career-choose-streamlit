## 📌 Overview

**Career Intelligence Ecosystem** is an interactive web application built using Streamlit that helps users discover the most suitable career paths based on their skills, personality traits, and preferences.

The app analyzes multiple dimensions such as:

* Technical skills
* Personality traits
* Work environment
* Salary expectations

It then generates **data-driven career recommendations** with insights, visualizations, and personalized learning paths.

---

## ✨ Features

### 🏆 Smart Career Recommendations

* AI-inspired scoring system for career matching
* Top career suggestions ranked by compatibility
* Skill gap analysis for improvement

### 🕸️ Capability Radar

* Interactive radar chart using Plotly
* Multi-dimensional comparison of career fit

### 🗺️ Upskill Maps

* Personalized learning roadmap
* Structured course suggestions for missing skills

### 📈 Market Insights

* Demand analysis of careers
* Visual charts for better decision-making
* Export results as CSV

---

## 🧠 How It Works

The app uses a **custom scoring algorithm** that evaluates:

* Skill match (45%)
* Personality traits (25%)
* Work environment (15%)
* Salary compatibility (15%)

It then ranks careers based on total compatibility score.

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/career-intelligence-app.git
cd career-intelligence-app
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
streamlit run app.py
```
